## What is CodeMeYo?

CodeMeYo is a native desktop application that acts as a fully autonomous coding agent. Give it a task — it reads your codebase, writes and edits files, installs dependencies, runs builds, fixes errors, and commits. All on its own.

It integrates multiple frontier LLMs — **Claude**, **GPT**, **Gemini**, **DeepSeek**, **Grok**, and **Ollama** — through a unified adapter system with full tool use and function calling support.

Built with **Tauri 2** (Rust backend + React frontend), CodeMeYo runs natively on all major desktop platforms with minimal resource usage and full local data privacy.

---

## Key Highlights

- **Autonomous agent loop** — planning, tool execution, error recovery, and verification
- **Six LLM providers** — Claude, GPT, Gemini, DeepSeek, Grok, Ollama
- **Multi-model modes** — Single Agent, Round Robin, Deep Think, Consensus, Fallback
- **MCP server integration** — extend the agent with external tools via the Model Context Protocol
- **Browser automation** — CDP deep debugging, extension bridge, live DOM inspection
- **Full development toolkit** — Monaco editor, xterm.js terminal, file explorer, git panel
- **Privacy-first** — all data stays local, API keys stored in your OS keychain
