# CodeMeYo

**Multi-LLM Autonomous Coding Agent**

A free desktop application that writes, edits, and runs code autonomously — powered by Claude, GPT, Gemini, DeepSeek, Grok, and more.

Available for **Windows**, **macOS**, and **Linux**.

## Download

Get the latest release from the [Releases](https://github.com/jagjourney/codemeyo/releases) page.

| Platform | Formats |
|----------|---------|
| Windows | `.msi`, `.exe` |
| macOS | `.dmg` |
| Linux | `.AppImage`, `.deb`, `.rpm` |
