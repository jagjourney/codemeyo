# CodeMeYo

**Multi-LLM Autonomous Coding Agent**

A free desktop application that writes, edits, and runs code autonomously. Powered by Claude (Anthropic), GPT (OpenAI), and Grok (xAI).

Available for **Windows**, **macOS**, and **Linux**.

## Features

- **Multi-LLM Support**: Claude (Anthropic), GPT (OpenAI), Grok (xAI)
- **4 Agent Modes**: Single, Round-Robin, Fallback, Deep Think (multi-LLM consensus)
- **27 Built-in Tools**: File ops, git, shell commands, browser debugging (CDP + Extension)
- **Monaco Editor**: Full code editor with diff view (same engine as VS Code)
- **MCP Server Integration**: Connect to databases, APIs, and custom tools
- **Browser Automation**: 3-phase system with DOM access, console logs, network monitoring
- **Project Indexing**: Smart codebase analysis with auto-detection and context injection
- **Screenshot Capture**: Full screen or region capture for visual context
- **Integrated Terminal**: Built-in terminal (xterm.js) for command execution
- **Secure Keychain**: API keys stored in your OS keychain, never in plain text
- **100% Local**: All data stored on your machine. No telemetry. You own everything.
